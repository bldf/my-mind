# M0-M4 Performance Report

## Targets

| Scenario | Target |
| --- | --- |
| 100 node initial render | P95 < 1s |
| 500 node validation/search | No visible blocking |
| 1000 node browse/search/fold/zoom | Usable in beta with documented risk |
| Core gzip | < 80 KB |
| React gzip | < 160 KB |

## Current Local Method

- Fixtures are generated by `pnpm fixtures`.
- Parse/search smoke bench runs with `pnpm bench`.
- Browser render, drag, and export checks are covered by `tests/e2e/playground.spec.ts`.

## Beta Risk Notes

- PNG export depends on browser Canvas and should be tested in real browsers.
- Worker layout scheduling is in place; ELK-quality layout can be swapped in behind the same graph contract.
