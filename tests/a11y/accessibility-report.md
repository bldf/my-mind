# Accessibility Report

## Checks

- Toolbar buttons use icon buttons with `aria-label` and `title`.
- Node title inputs have accessible labels.
- Theme state is not expressed by color alone; selected theme has a check icon.
- Task state uses visible text labels.
- Default text colors in built-in themes target WCAG AA contrast.

## Manual Keyboard Path

1. Focus the editor container.
2. Select a node.
3. Use `Tab`, `Enter`, `Delete`, `Ctrl/Cmd+Z`, and `Ctrl/Cmd+Shift+Z`.
4. Open theme, search, and inspector controls from the toolbar.
