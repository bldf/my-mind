# FAQ

## Does the library upload user data?

No. The packages operate on local data and do not include a backend service.

## Is SSR safe?

`@my-mind-node/core`, `@my-mind-node/importers`, and text exporters are SSR safe.
React canvas rendering and PNG export require a browser.

## What is experimental?

Provider integrations, collaboration adapters, plugin contracts, and advanced
mobile drag editing remain experimental during beta.
